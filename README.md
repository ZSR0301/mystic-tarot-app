# 🔮 Mystic Tarot - 交互式塔罗牌占卜应用

一个基于Web的交互式塔罗牌占卜应用，结合了手势识别和3D视觉效果，提供沉浸式的塔罗牌体验。

## ✨ 特性

- 🎴 **完整塔罗牌组**：包含78张塔罗牌（大阿卡纳和小阿卡纳）
- 🖐️ **手势控制**：支持MediaPipe手势识别和鼠标控制两种交互方式
- 🎨 **3D视觉效果**：使用Three.js实现的精美3D卡牌展示和粒子特效
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🌟 **卡牌解读**：每张卡牌都有详细的正位和逆位含义解释
- 📜 **历史记录**：保存占卜历史，方便回顾

## 🚀 快速开始

### 在线体验

直接在浏览器中打开 `tarot.html` 文件即可开始体验。

### 本地运行

1. 克隆或下载此仓库
2. 使用本地服务器打开项目（推荐使用Live Server等插件）
3. 允许摄像头权限以获得最佳体验（可选）

## 🎮 操作说明

### 手势控制模式

- **五指张开**：浏览/轮转模式，左右挥动手掌可轮换卡牌
- **食指指向**：瞄准模式，移动食指选择卡牌
- **捏合手势**：抓取/确认模式，保持捏合1秒或向下拖拽确认选择
- **双手张开**：超级轮转模式，快速挥动双手进行高速轮转

### 鼠标控制模式

- **鼠标移动**：选择卡牌
- **左键按住拖动**：轮换卡牌
- **右键点击**：确认选择
- **双击**：查看卡牌详情

## 🛠️ 技术栈

- **前端框架**：原生HTML/CSS/JavaScript
- **3D渲染**：Three.js
- **手势识别**：MediaPipe Hands
- **卡牌数据**：基于[tarot-json](https://github.com/ekelen/tarot-json)项目
- **样式设计**：现代CSS3效果和动画

## 🎯 卡牌布局

应用采用扇形展开布局，卡牌以优雅的弧形排列在屏幕底部。选中的卡牌会高亮显示并上浮，提供清晰的视觉反馈。

## 🌟 特殊效果

- **粒子系统**：卡牌轮转和选择时的动态粒子效果
- **翻转动画**：卡牌从背面翻转到正面的平滑过渡
- **消失效果**：选中卡牌后的渐隐消失动画
- **手势指示器**：实时显示当前手势状态的视觉反馈

## 📝 占卜流程

1. 选择3张卡牌代表过去、现在和未来
2. 每张卡牌会显示在顶部的展示框中
3. 点击卡牌可查看详细解读
4. 占卜结果会保存在左侧的历史记录中
5. 点击"新一轮"按钮开始新的占卜

## 🔧 自定义配置

可以通过修改 `tarot.html` 中的 `CONFIG` 对象来自定义：

- 卡牌路径
- 卡牌数量
- 动画速度
- 粒子效果参数
